The site is organized into three main areas, accessible from the top navigation bar:

- **Home** — a summary dashboard showing current ED trends for all three viruses, plus a neighborhood-level map and respiratory panel results.
- **Data** — detailed charts and tables for a specific virus. Use the left sidebar to switch viruses and data types.
- **About** — background on data sources, methodology, and how to use the site.

#### Switching viruses and data types

On any Data page, the left sidebar has two controls:

- **Virus** — select COVID-19, Flu, or RSV to load data for that illness.
- **Data type** — choose how illness activity is measured: *Emergency Department visits*, *Lab Cases*, *Deaths* (COVID-19 only), or *Wastewater* (COVID-19 only).

> On mobile, tap the **☰ sliders icon** at the top left to open the sidebar controls.

#### Jump to a section

The **Jump to** links in the sidebar scroll you directly to a specific chart. Hover any link on desktop to see a live preview of that section's data before jumping.
